MARKETMIND - a chatbot for stock market users

Currently developing a Generative AI-based chatbot designed to provide stock market insights through natural language interactions. The chatbot integrates Finnhub API for real-time financial data and leverages powerful LLMs like Gemma 2 9B and Llama 3.3 70B for query understanding and analysis. Built using Streamlit, the project aims to create an intuitive tool that simplifies market research and supports informed decision-making for investors.
